Show RSS Feed inside a post.

== Description ==

Using this plugin you can show RSS Feed inside any post. The RSS will be loaded when post is loaded so you can show dynamic relevant contents through RSS feeds. 

This plugin uses WordPress's built in feed handling functions that also uses cache. 

Read details on http://hungrycoder.xenexbd.com/?p=865

== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface or upload to wp-content/plugins directory using FTP.
1. Activate the plugin once uploaded from the wordpress plugin administration page.
1. Configure from the setting page (check screenshot)
1. You will get a Meta Box in post editing interface to mention Feed URL and Title.   (check screenshot)
1. You'r done.
